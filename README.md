# Homework 1

*This homework involves 3 questions:*

Q1: Write a Dart function that receives a number (either 0 or 1) and returns the other.

Q2: Write a Dart function that receives a string and checks whether the string contains all English
letters or not (26 letters).

Q3: Write a Dart function that receives a string, counts the occurrences of each letter, and then
prints the resulting map.


**The answer for each question is located in the bin and lib folder.**
